### 2. Voeg een Link naar de Live Site Toe

Er zijn twee manieren om een link naar je live site toe te voegen:

#### Methode 1: Repository Description (Aanbevolen)

1. Ga naar je repository: https://github.com/benkogerrie/privearts
2. Klik op het **tandwiel icoon** (âš™ï¸) naast "About" rechtsboven
3. Vink **"Website"** aan
4. Vul je Cloudflare Pages URL in: `https://privearts.pages.dev`
5. Klik op **"Save changes"**

Nu verschijnt er een link naar je live site rechtsboven in de repository!

#### Methode 2: README.md (Al gedaan)

De README.md heeft al een link naar de live site bovenaan. Als je je Cloudflare Pages URL hebt, vervang dan `https://privearts.pages.dev` met je eigen URL.

### 3. Vind je Cloudflare Pages URL

1. Ga naar https://dash.cloudflare.com
2. Klik op **"Pages"** in de linker navigatie
3. Klik op je project (bijv. `privearts`)
4. Je ziet de URL bovenaan, bijvoorbeeld: `https://privearts.pages.dev`

### 4. Custom Domain (Optioneel)

Als je een eigen domein hebt (bijv. `prive-arts.nl`):

1. Ga naar je Cloudflare Pages project
2. Klik op **"Custom domains"**
3. Voeg je domein toe
4. Volg de DNS instructies
5. Update de link in de README en repository description
